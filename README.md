# CS-Study

- 스터디 주제 및 목표
    - 각자 cs 관련한 주제에 대해 자료를 정리하여 발표하는 스터디
    - cs 관련 지식 습득 및 면접에 도움이 되는 것으로 목표
- 스터디 방식
    - 이주일동안 주제에 관련하여 공부하고 자료를 정리하여 격주 목요일에 발표
    - 발표 3일전까지 깃에 발표용 문서 올리고 각자 서로의 문서 확인 및 질문사항 체크
    - 최소 10분 이상 자신이 조사한 주제에 대한 발표 진행
    - 발표 후 주제에 대해 자유롭게 질문하고 피드백을 통해 내용을 추가하고 수정
- 발표 규칙 및 진행 방식
    - 스터디 전까지 별도의 연락없이 불참 시 벌금/벌칙 있음. 불가피한 사정이 있다면 미리 연락 → 개인사정으로 발표가 어려울 경우 각자 자료만 주고받는 걸로 진행
    - 한달에 최소 한번 스터디 참여
- 스터디 일정
    - 격주 목요일, 온라인(구글) 발표 진행
- 커리큘럼
    - 상세 주제는 다음과 같음
    - 네트워크, 운영체제, 자료구조, 데이터베이스, 디자인패턴, 알고리즘, web
